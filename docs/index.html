Simple Java Flight Simulator

Control the aircraft’s speed, altitude, and heading in this basic Java flight simulator built using JavaFX.

About the Flight Simulator

This is a simple flight simulator written in Java. It allows you to control an aircraft’s speed, altitude, and heading using keyboard controls. It’s built using JavaFX for graphical rendering.

Controls

  • Up Arrow - Increase altitude
  • Down Arrow - Decrease altitude
  • Left Arrow - Turn left (heading counter-clockwise)
  • Right Arrow - Turn right (heading clockwise)
  • W - Increase speed
  • S - Decrease speed

How to Run the Flight Simulator

To run the simulator, follow these steps:

  1. Ensure you have Java 11 or later installed.
  2. Download and set up JavaFX.
  3. Compile and run the FlightSim.java file.
